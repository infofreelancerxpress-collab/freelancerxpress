# FreelancerXpress

FreelancerXpress is a modern web application built with Next.js, designed to connect clients with skilled freelancers for various services. It features a comprehensive set of tools for managing projects, services, and client-freelancer interactions.

## Features

- **Service Listings**: Browse and discover a wide range of services offered by freelancers.
- **Influencer Marketing**: Dedicated section for influencer marketing services with detailed profiles.
- **Contact Form**: Easy communication channel for inquiries.
- **Responsive Design**: Optimized for various devices using Tailwind CSS.
- **Interactive UI**: Powered by Radix UI and Framer Motion for a smooth user experience.
- **Type-safe Development**: Built with TypeScript for robust and maintainable code.

## Technologies Used

- **Framework**: Next.js
- **Frontend**: React
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Animations**: Framer Motion
- **Form Validation**: Zod
- **Email Services**: Nodemailer
- **Language**: TypeScript

## Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v18 or higher)
- bun (or npm/yarn)

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd freelancerxpress
   ```
2. Install dependencies:
   ```bash
   bun install
   ```

### Running the Development Server

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js application routes and API endpoints.
- `components/`: Reusable UI components.
  - `components/ui/`: Shadcn UI components.
- `data/`: Mock data or static content.
- `hooks/`: Custom React hooks.
- `lib/`: Utility functions.
- `public/`: Static assets.
- `types/`: TypeScript declaration files.
